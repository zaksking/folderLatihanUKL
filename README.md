Fungsi Program:

Program ini digunakan untuk menghitung biaya pengiriman paket ekspedisi berdasarkan berat barang, jarak tempuh, dan volume barang.
Program juga menambahkan biaya tambahan jika volume barang melebihi batas tertentu (100 cm³).

Cara Kerja Program:

Program ini bekerja dengan meminta pengguna memasukkan data panjang, lebar, tinggi, berat, dan jarak barang, lalu menghitung volume barang serta menentukan tarif berdasarkan jarak. Jika jarak kurang dari atau sama dengan 10 km, tarifnya Rp 4250 per kilogram, sedangkan jika lebih dari 10 km, tarifnya Rp 6000. Jika volume barang melebihi 100 cm³, maka ditambahkan biaya tambahan sebesar Rp 50.000. Setelah itu, program menghitung total biaya pengiriman dari hasil perkalian berat dan tarif ditambah biaya tambahan, kemudian menampilkan hasil akhirnya ke layar.

 Fungsi Program:

Program ini berfungsi untuk menentukan apakah suatu bilangan yang dimasukkan oleh pengguna merupakan bilangan genap atau bilangan ganjil.

Cara Kerja Program:

Program meminta pengguna memasukkan sebuah bilangan, kemudian memeriksa sisa hasil bagi bilangan tersebut dengan angka 2 menggunakan operator modulus (%). Jika hasilnya 0, program menampilkan bahwa bilangan tersebut adalah genap, dan jika tidak 0 maka bilangan tersebut ganjil. Setelah  itu, hasilnya ditampilkan ke layar.

Soal No 3 (sedang 1) 

<img width="319" height="68" alt="image" src="https://github.com/user-attachments/assets/0c1d1d0b-ef2b-474f-9bf7-42a45772e559" />

Fungsi Program:

Program ini digunakan untuk menghitung faktorial dari bilangan bulat positif yang dimasukkan oleh pengguna. Faktorial sendiri adalah hasil perkalian semua bilangan dari 1 sampai n.

Cara Kerja Program:

Program meminta pengguna memasukkan bilangan bulat positif. Jika bilangan yang dimasukkan negatif, maka program akan langsung berhenti dan tidak berjalan. Jika bilangan valid, program akan menggunakan perulangan for untuk menghitung faktorial dari 1 hingga n dengan bantuan tipe data BigInteger, agar hasilnya bisa menampung angka yang sangat besar. Setelah selesai menghitung, program menampilkan hasil faktorial ke layar.

Fungsi Program:

Program ini berfungsi untuk menghitung volume tabung berdasarkan jari-jari alas dan tinggi yang dimasukkan oleh pengguna.

Cara Kerja Program:

Program meminta pengguna memasukkan nilai jari-jari dan tinggi tabung. Nilai tersebut kemudian dikirim ke fungsi hitungTabung, yang menghitung volume menggunakan rumus π × r × r × t. Hasil perhitungan dikembalikan ke program utama, lalu ditampilkan ke layar sebagai hasil perhitungan volume tabung.

Soal no 5 (sulit 1)

Fungsi Program:

Program ini digunakan untuk menghitung jumlah total nilai dan rata-rata nilai siswa berdasarkan jumlah siswa yang dimasukkan oleh pengguna.

Cara Kerja Program:

Program meminta pengguna memasukkan jumlah siswa terlebih dahulu. Lalu, menggunakan perulangan while, program meminta pengguna menginput nilai setiap siswa satu per satu. Nilai yang dimasukkan dijumlahkan ke dalam variabel totalNilai. Setelah semua nilai dimasukkan, program menghitung rata-rata dengan membagi total nilai dengan jumlah siswa, kemudian menampilkan hasil jumlah dan rata-rata nilai tersebut ke layar.

Soal no 6 (sulit 2)

<img width="236" height="279" alt="image" src="https://github.com/user-attachments/assets/9c0fc71c-7960-4178-8bfc-3ffd695526f5" />

Fungsi Program:

Program ini berfungsi untuk melakukan penjumlahan dua buah matriks dengan ukuran yang sama, yaitu berdasarkan jumlah baris dan kolom yang dimasukkan oleh pengguna.

Cara Kerja Program:

Program meminta pengguna memasukkan jumlah baris dan kolom matriks, kemudian menginput nilai-nilai elemen untuk matriks A dan matriks B. Setelah semua elemen dimasukkan, program menjumlahkan elemen yang memiliki posisi sama dari kedua matriks tersebut dan menyimpannya ke dalam matriks C. Terakhir, program menampilkan hasil penjumlahan matriks A dan B dalam bentuk matriks C ke layar.
